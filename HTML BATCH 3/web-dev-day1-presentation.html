What is the Internet? 🌍

The Internet is a global network of connected computers that communicate with each other.

  • Think of it as millions of roads connecting cities 🛣️
  • Uses cables, satellites & wireless signals 📡
  • Enables email, video calls, file sharing & more
  • Existed before the World Wide Web

Analogy: Internet = the highway system 🚗

What is the Web? 🕸️

The Web (World Wide Web) is a collection of websites and web pages accessible via the Internet.

  • Uses browsers like Chrome, Firefox, Edge 🌐
  • Built with HTML, CSS & JavaScript
  • Connected through hyperlinks (clickable links)
  • One of many services that run ON the Internet

Analogy: Web = the shops & buildings along the highway 🏪

What is WWW? 🔗

WWW stands for World Wide Web — invented in 1989 at CERN.

  • A system of interlinked hypertext documents
  • Accessed via URLs (web addresses)
  • Uses the HTTP protocol to transfer data
  • The "www" you see in URLs refers to this system

🌐 www.example.com — the "www" is a subdomain pointing to the web server!

Founder of the WWW 👨‍💻

Sir Tim Berners-Lee invented the World Wide Web in 1989 while working at CERN, Switzerland.

  • Created the first web browser & server
  • Designed HTML, HTTP & URLs
  • Made the Web free for everyone 🎁
  • Founded the W3C to develop web standards

🏆 He received the Turing Award in 2016 — the "Nobel Prize of Computing"

What is HTML? 📄

HTML = HyperText Markup Language

  • The skeleton of every web page 🦴
  • Uses tags like <h1>, <p>, <img>
  • Defines headings, paragraphs, images, links
  • NOT a programming language — it's a markup language!

🏠 Analogy: HTML is like the walls, doors & rooms of a house

What is HTTP? 📨

HTTP = HyperText Transfer Protocol

  • The language browsers & servers use to talk 🗣️
  • When you visit a URL, browser sends an HTTP Request
  • Server responds with an HTTP Response (the webpage!)
  • HTTPS = secure version (with encryption 🔒)

📬 Analogy: HTTP is like the postal service delivering your mail!

Internet VS Web 🆚

🌍 Internet

The physical network infrastructure — cables, routers, servers connecting computers worldwide

≠

🕸️ Web

A service that runs ON the Internet — websites, pages, and apps you browse daily

💡 Email, online gaming & video streaming use the Internet but are NOT the Web!

What is a Browser? 🖥️

A web browser is software that displays web pages.

  • Chrome — by Google 🔵
  • Firefox — by Mozilla 🦊
  • Edge — by Microsoft 🔷
  • Safari — by Apple 🍎

🪟 Analogy: Browser = the window you look through to see websites

What is a Search Engine? 🔍

A search engine is a website that helps you find information on the Web.

  • Google — most popular worldwide
  • Bing — by Microsoft
  • DuckDuckGo — privacy-focused
  • Crawls billions of web pages & ranks results

📚 Analogy: Search Engine = a librarian who finds books for you!

Browser VS Search Engine 🆚

🖥️ Browser

Software installed on your computer. Displays ANY website — YouTube, Facebook, Google, etc.

≠

🔍 Search Engine

A website you visit INSIDE a browser. Helps you search & find other websites.

🧠 Key Point: You use a Browser to OPEN a Search Engine. Chrome is NOT Google — Chrome is the browser, Google is a website!
