3> Types of CSS :

1. Inline CSS

in this we use Style Attribute (style="")
>>this has the highest priority for changes

uses of Inline CSS:

>> To write CSS for any perticular line or tag of HTML.
>> For Testing purpose

Recommendation:

It is not Recommended for large projects.


2> Internal CSS

we use Style Tag to write CSS for any perticular HTML page
this has the least priority for changes
uses:

>> when writin css for any perticular web page

Recommendation:

It is only usedul for Small websites

3> External CSS

a file created with .css ([DOT]css) extention, and we write css inside that file
