HTML boilerplate + HTML tree

Root element, head/meta tags, and body tags.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello Duniya</h1>
  </body>
</html>
HTML tree (visual)
html (root)
├─ head
│  ├─ meta charset
│  ├─ meta viewport
│  └─ title
└─ body
   └─ visible content
`html` root `head` `body` visible
